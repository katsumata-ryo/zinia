# RESEARCH — 年末調整 制度リサーチメモ（一次情報）

> このファイルは書籍『年末調整を体系的に理解する』の各章執筆者が参照する一次情報メモです。
> 金額・要件・様式は必ず本メモの出典URL（国税庁 nta.go.jp 中心）に当たり直してから本文へ反映してください。
> 調査日: 2026-07-10 時点。制度は毎年改正されるため、執筆時点で再確認すること。

## 0. 成立状況の整理（重要）

- **令和7年度税制改正**: 成立・公布済み。原則 **令和7年12月1日施行**、**令和7年分以後**の所得税に適用。
  国税庁が「見直し等が行われました」と過去形で告知しており、確定情報として扱ってよい。
- **令和8年度税制改正**: 元は令和7年12月26日閣議決定の「令和8年度税制改正の大綱」。
  その後、国税庁 2026kiso ページが「基礎控除の引上げ等が行われました」「令和8年12月1日に施行され、令和8年分以後の所得税について適用されます」と過去形で告知しているため、**成立済みの前提で記述してよい**。
  ただし本の陳腐化対策として、各章では「令和◯年分時点」注記（`.year-note`）を必ず添えること。
  MOF大綱（財務省 20251226taikou.pdf）はあくまで大綱段階の資料である点に留意。

## 1. 基礎控除（令和7年改正）

合計所得金額に応じた控除額（令和7年分）。旧制度は一律48万円だった。

| 合計所得金額 | 基礎控除額（R7分） | 旧 |
|---|---|---|
| 132万円以下 | **95万円** | 48万円 |
| 132万円超 336万円以下 | 88万円 | 48万円 |
| 336万円超 489万円以下 | 68万円 | 48万円 |
| 489万円超 655万円以下 | 63万円 | 48万円 |
| 655万円超 2,350万円以下 | 58万円（本則） | 48万円 |
| 2,350万円超 2,400万円以下 | 48万円 | 48万円 |
| 2,400万円超 2,450万円以下 | 32万円 | 32万円 |
| 2,450万円超 2,500万円以下 | 16万円 | 16万円 |
| 2,500万円超 | 0円 | 0円 |

- 132万円超〜655万円以下の上乗せ（88/68/63万円）は**令和7・8年分限定の特例**という位置づけ。令和9年分以後は段階的に本則58万円へ調整される予定。→ 適用年の区別が重要。
- 出典: [令和7年度税制改正による所得税の基礎控除の見直し等について](https://www.nta.go.jp/users/gensen/2025kiso/index.htm)、[源泉所得税の改正のあらまし PDF](https://www.nta.go.jp/publication/pamph/gensen/0025005-051.pdf)

## 2. 給与所得控除（令和7年改正）

- **最低保障額を 55万円 → 65万円 に引上げ**（令和7年分〜）。
- 出典: 上記 2025kiso ページ、[No.1199 基礎控除](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/1199.htm)

## 3. 特定親族特別控除（令和7年創設）

- 居住者が**特定親族**（年齢19歳以上23歳未満、合計所得金額 **58万円超123万円以下**。里子含む。配偶者・青色/白色事業専従者は除く）を有する場合、特定親族1人につき合計所得金額に応じ**最高63万円**を控除。
- 従来の特定扶養控除（63万円）が使えなくなる所得帯（合計所得58万円超）をなだらかに埋める制度。いわゆる「大学生年代のバイト収入の壁」対策。
- 年末調整では「給与所得者の特定親族特別控除申告書」の提出が必要。
- 月次源泉徴収での考慮対象は、給与のみなら合計所得58万円超100万円以下。
- 出典: [2025kiso index](https://www.nta.go.jp/users/gensen/2025kiso/index.htm)、[A2-4 特定親族特別控除の申告](https://www.nta.go.jp/taxes/tetsuzuki/shinsei/annai/gensen/annai/1648_73.htm)

## 4. 扶養親族等の所得要件（令和7年改正）

- 扶養親族・同一生計配偶者の**合計所得金額要件: 48万円以下 → 58万円以下**（給与収入のみなら103万円→**123万円**に相当）。
- 勤労学生: 合計所得**85万円以下**（旧75万円以下）に引上げ。
- 出典: 2025kiso ページ、[令和7年分 扶養控除額の早見表 PDF](https://www.nta.go.jp/publication/pamph/gensen/nencho2025/pdf/117.pdf)

## 5. 令和8年度改正（令和8年分以後）

- **基礎控除の本則を 58万円 → 62万円 に引上げ**（合計所得655万円超2,350万円以下の標準額。物価スライドで+4万円）。
- **給与所得控除の最低保障額**: 本則は65万円→69万円だが、**令和8・9年分は特例で 74万円**。
- **課税最低限 178万円**（＝特例の基礎控除104万円相当＋給与所得控除特例74万円）に。いわゆる「103万円の壁」→令和7年で160万円、令和8年で**178万円**へ。
  ※「160万円/178万円の壁」は住民税・社保の壁と混同されやすいので第18章で丁寧に切り分けること。
- **扶養親族等の合計所得金額要件: 58万円 → 62万円 以下**（給与収入のみ相当で123万円→引上げ）。
- **適用時期**: 令和8年12月1日施行、令和8年分以後。ただし**月次の源泉徴収税額表は令和8年中は改正されず（令和9年分の税額表から改正）**、令和8年分は差額を**年末調整で精算**する。→ 第22章の要点。
- 物価スライド制（物価上昇に連動して基礎控除等を引き上げる仕組み）が導入された点も第23章の論点。
- 出典: [令和8年度税制改正による所得税の基礎控除の引上げ等について](https://www.nta.go.jp/users/gensen/2026kiso/index.htm)、[源泉所得税の改正のあらまし（令和8年5月）PDF](https://www.nta.go.jp/users/gensen/2026kiso/pdf/0026005-024.pdf)、[令和8年度税制改正の大綱（財務省）](https://www.mof.go.jp/tax_policy/tax_reform/outline/fy2026/20251226taikou.pdf)

## 6. 申告書様式の変更（令和8年分）

- 令和8年分の扶養控除等（異動）申告書から、記載区分が「控除対象扶養親族」中心から **「源泉控除対象親族」** を記載する形に変更。
  - **源泉控除対象親族** = 控除対象扶養親族 または 特定親族 のうち、合計所得金額が **100万円以下** の者（＝月次源泉で扶養親族等の数にカウントできる親族）。
- 各種申告書は「基礎控除申告書 兼 配偶者控除等申告書 兼 特定親族特別控除申告書 兼 所得金額調整控除申告書」という統合様式。実務略称の対応（マル扶／マル保／マル基配所）を用語集で整理。
- 出典: [各種申告書・記載例](https://www.nta.go.jp/users/gensen/nencho/shinkokusyo/index.htm)、[令和8年分の給与の源泉徴収事務 PDF](https://www.nta.go.jp/publication/pamph/gensen/nencho2025/pdf/113.pdf)、[令和8年分 扶養控除等申告書 記載のしかた PDF](https://www.nta.go.jp/publication/pamph/gensen/shikata_r08/pdf/05.pdf)

## 8. 執筆時の注意（陳腐化・正確性）

1. 金額を書くときは必ず「令和◯年分」を明記し、`.year-note` を付ける。
2. 令和7年分と令和8年分で基礎控除・給与所得控除・所得要件が異なる。計算例はどの年分か明示する。
3. 特例（時限措置）と本則を区別する。令和9年分以後の逓減にも触れる。
4. 復興特別所得税（基準所得税額×2.1%）は令和19年まで継続。年調年税額の算出で必須。
5. 未確認の詳細数値（各控除の速算表、所得金額調整控除の細目、生命保険料控除の新旧計算式）は各章執筆時に一次情報で補完すること。本メモは骨子のみ。
